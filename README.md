# Client Follow-Up Tracker

A lightweight, browser-based CRM for tracking client leads and follow-ups.
Built with plain HTML, CSS, and vanilla JavaScript. No frameworks, no backend.
Data is stored locally in the browser via localStorage.

## Who it's for
Small teams who need a simple way to track leads without a full CRM setup.

## Features (v1)
- Add, edit and view client contacts
- Track lead status through a pipeline
- Set next follow-up dates
- Log timestamped notes per contact
- Data persists in the browser
